Qu'est ce que le développement web ?

Le développement Web fait généralement référence aux tâches liées au développement de sites Web conçus pour être hébergés sur Internet. Le processus de développement Web comprend, mais sans s'y limiter, la conception de sites Web, le développement de contenu Web, les scripts côté client ou côté serveur et la configuration de la sécurité du réseau.
